# KBC Quiz App

A KBC-style Quiz Application developed using Flutter and Firebase with interactive UI, timer functionality, lifelines, leaderboard, and real-time quiz features.

## Features
- 50-50 Lifeline
- Audience Poll
- Timer Functionality
- Real-Time Leaderboard
- Firebase Authentication
- Category-wise Questions
- REST API Integration
- Responsive UI

## Tech Stack
- Flutter
- Firebase
- REST API
- Provider / Riverpod
- Firestore
